# woo — user reference

Task-oriented reference for people **using** a woo world. For protocol
and substrate details, see [`SPEC.md`](../SPEC.md). For contributor
guidance, see [`AGENTS.md`](../AGENTS.md).

## By role

- **[wizard/](wizard/)** — operations that require wizard authority:
  recycle, force operations, session management, world-level state.

Sections for player and builder reference will land alongside their first
pages; until then, the in-world `help` system is the canonical guide for
those audiences.

## Conventions

- `;…` denotes evaluation in the in-world `eval` surface (the `$programmer:eval`
  verb, or the equivalent in your client).
- `$foo` denotes a corename — a stable, human-readable handle for an
  object.
- Code blocks show woocode (the in-world DSL), not TypeScript.
